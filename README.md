# To-Do List Manager

This is a simple Bash scripting project that allows users to manage tasks from the terminal.

---

# Features

- View tasks
- Add tasks
- Delete tasks
- Save tasks in ~/todo.txt

---

# How To Run

```bash
chmod +x todo.sh
./todo.sh
```
